# Yunlin Zombie Arsenal Weapon Plan

## Theme
Zombie apocalypse survival weapons. All weapons are fictional.

## Weapons

1. Rustline-9 Pistol
- Type: pistol
- Role: early-game backup weapon
- Ammo: 9mm
- Style: worn metal, taped grip

2. Wardog Rifle
- Type: rifle
- Role: main mid-range weapon
- Ammo: 5.56 or custom rifle ammo
- Style: scavenged military rifle

3. Longwatch Sniper
- Type: sniper
- Role: long-range high damage
- Ammo: 7.62 or custom sniper ammo
- Style: long barrel, scope, cloth wrap

4. Scrapper-12 Shotgun
- Type: shotgun
- Role: close-range zombie clearing
- Ammo: shotgun shells
- Style: rusty short shotgun

5. Bunker HMG
- Type: heavy machine gun
- Role: late-game heavy weapon
- Ammo: ammo box / belt ammo
- Style: bulky, heavy, slow
